# Tickets — editor de opções por select

- `ticketedit > Opções do painel` agora mostra uma opção selecionada em vez de abrir modal direto ao selecionar.
- Adicionado botão cinza `✏️ Editar` para editar a opção selecionada.
- Adicionado botão cinza `➕` para criar opção personalizada sem usar checkbox no modal de opções.
- O modal `Editar opções` agora só liga/desliga opções existentes e configura múltiplos tickets, transcript e webhook.
- O fluxo da opção deixou de ser digitado manualmente e virou select em modal V2 quando disponível.
- Mantido fallback por TextInput para versões/libs que ainda não aceitam select/channel select em modal.
