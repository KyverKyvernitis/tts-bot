# Patch tickets — editor simplificado

- Simplifica o `ticketedit` para um resumo curto.
- Troca os botões grandes do editor por um único select de edição.
- Mantém só dois botões no editor: `Preview` e `⛔`.
- Remove o botão manual de atualizar painel publicado.
- Remove o botão textual `Apagar editor`.
- Atualiza automaticamente o painel publicado após salvar configurações, quando existir painel salvo.
